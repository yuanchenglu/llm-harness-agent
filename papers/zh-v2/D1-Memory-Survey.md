# D1. 基于 LLM 智能体的记忆机制综述

**标题：** A Survey on the Memory Mechanism of Large Language Model Based Agents
**作者：** Zeyu Zhang 等
**出处：** ACM TOIS, Vol. 43(6), 2025（CCF-A 期刊/SCI）
**编号：** arXiv:2404.13501

---

**核心：** 从三个视角系统梳理记忆机制：(1) 结构——感觉/短期/长期/情景/程序性记忆；(2) 操作——存储、检索、巩固、遗忘；(3) 整合模式——记忆模块如何与 LLM 核心交互。涵盖 100+ 篇论文。

**Harness 启示：** 直接对应装备层模型的状态存储（S）和上下文管理器（C）。核心发现——记忆结构和检索机制独立于底层 LLM 显著影响性能——支持"记忆即基础设施"论。
