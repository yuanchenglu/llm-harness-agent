# DeepSeek-V3.2：推动开源大语言模型的前沿

**作者：** DeepSeek-AI
**论文编号：** arXiv:2512.02556v1

---

## 摘要

我们推出 **DeepSeek-V3.2**，一个将高计算效率与卓越推理和 Agent 性能相统一的模型。关键技术突破：(1) **DeepSeek Sparse Attention (DSA)**——高效注意力机制，大幅降低计算复杂度同时保持长上下文性能；(2) **FP8 低精度训练**的进一步优化——实现更高训练效率；(3) **强化学习扩展**——通过大规模 RL 进一步提升推理能力；(4) **Agent 能力增强**——改进工具使用和函数调用能力。DeepSeek-V3.2 在标准基准上超越之前版本，同时在推理和 Agent 任务上达到与领先闭源模型相当的性能。
