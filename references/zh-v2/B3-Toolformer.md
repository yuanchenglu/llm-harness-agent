# B3. Toolformer：语言模型可以自学使用工具

**标题：** Toolformer: Language Models Can Teach Themselves to Use Tools
**作者：** Timo Schick 等（Meta AI）
**出处：** NeurIPS 2023（CCF-A）
**编号：** arXiv:2302.04761

---

**核心贡献：** 提出自监督方法训练模型自主决定调用哪些 API、何时调用、传什么参数、如何融入结果。模型在文本中嵌入特殊标记的 API 调用，通过自监督学习——用 API 返回结果替代调用标记，筛选出"使后续文本预测更容易"的调用进行训练。

**技能：** 问答（维基百科）、计算器、翻译、日历。Toolformer 在零样本下游任务上显著优于同规模未增强模型，甚至与更大模型竞争。

**Harness 启示：** 工具调用不是模型能力，而是装备层协议——模型提议调用，装备层解析、执行和整合结果。
